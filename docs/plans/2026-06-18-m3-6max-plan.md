**实现:**

```go
// Package sidepot 实现多人德州扑克的边池(side pot)计算。
// 给定每个玩家本手总投入与是否弃牌,算出多级 pot 的层级与每层赢家集合。
package sidepot

import "sort"

// Pot 是一个边池层级。
type Pot struct {
    Amount      int    // 该层 pot 金额
    Eligible    []int  // 有资格争该层的玩家 seat 索引(按 seat 升序)
}

// Compute 按各玩家总投入(contrib[i])与是否争胜负(contending[i]=未弃牌)
// 计算多级边池。
//   contrib[i]   玩家 i 本手总投入(>= 0)
//   contending[i] 玩家 i 是否仍在争胜负(false=已弃牌,只贡献不争)
// 返回的 Pot 列表按层级升序(主池在前,边池在后)。零投入层被跳过。
func Compute(contrib []int, contending []bool) []Pot {
    if len(contrib) != len(contending) {
        panic("sidepot: contrib and contending length mismatch")
    }
    n := len(contrib)
    if n == 0 {
        return nil
    }
    // 收集去重升序的阈值(>0 的 contribution)
    levels := map[int]struct{}{}
    for _, c := range contrib {
        if c > 0 {
            levels[c] = struct{}{}
        }
    }
    if len(levels) == 0 {
        return nil
    }
    sorted := make([]int, 0, len(levels))
    for lv := range levels {
        sorted = append(sorted, lv)
    }
    sort.Ints(sorted)

    var pots []Pot
    prev := 0
    for _, lv := range sorted {
        diff := lv - prev
        if diff <= 0 {
            continue
        }
        // 该层贡献人数 = contrib >= lv 的所有玩家(包括弃牌者,他们的钱在池里)
        // 该层赢家资格 = contrib >= lv 且 contending 的玩家
        contribCount := 0
        var eligible []int
        for i := 0; i < n; i++ {
            if contrib[i] >= lv {
                contribCount++
                if contending[i] {
                    eligible = append(eligible, i)
                }
            }
        }
        amount := diff * contribCount
        if amount > 0 {
            pots = append(pots, Pot{Amount: amount, Eligible: eligible})
        }
        prev = lv
    }
    return pots
}

// Distribute 把 pots 按 winners(每层赢家 seat 索引列表)分发,
// 返回每人赢得的总额。余数(不能整除)给 eligible 中顺位最先的赢家。
//   pots[i].Eligible 必须按 seat 升序(Compute 保证)
//   winnersByPot[i] 必须是 pots[i].Eligible 的子集
// 若某层 winnersByPot[i] 为空(无人争,例如独贡献者),则该层金额
// 发给 Eligible 里顺位最先的(其实就是唯一贡献者)。
func Distribute(pots []Pot, winnersByPot [][]int) []int {
    if len(pots) != len(winnersByPot) {
        panic("sidepot: pots and winnersByPot length mismatch")
    }
    // 用最大 seat 索引确定 result 长度
    maxSeat := -1
    for _, p := range pots {
        for _, s := range p.Eligible {
            if s > maxSeat {
                maxSeat = s
            }
        }
    }
    result := make([]int, maxSeat+1)
    for i, pot := range pots {
        winners := winnersByPot[i]
        if len(winners) == 0 {
            // 无人争,发给 eligible 顺位最先
            if len(pot.Eligible) > 0 {
                result[pot.Eligible[0]] += pot.Amount
            }
            continue
        }
        share := pot.Amount / len(winners)
        rem := pot.Amount - share*len(winners)
        // winners 按 seat 升序排序,保证余数给顺位最先
        sort.Ints(winners)
        for j, w := range winners {
            add := share
            if j == 0 {
                add += rem
            }
            result[w] += add
        }
    }
    return result
}
```

**测试覆盖(7 个 case):**
1. Heads-up 等额(contrib=[10,10], contending=[t,t]) → 1 个 pot 20,2 人都 eligible
2. 3 人单 all-in(A=100, B=50, C=50 全 contending) → 3 层:50×3=150 主池(A,B,C),50×2=100 边池(A,B);无 100 独拿层(只有 A 投 100,但层级阈值是 100,diff=50,A 独拿 50)。仔细算:阈值[50,100],prev=0:lv=50,diff=50,contrib>=50 都三人=3 → pot 150,eligible [0,1,2]。prev=50:lv=100,diff=50,contrib>=100 只有 A → pot 50,eligible [0]。总 2 层 ✓
3. 含弃牌者:contrib=[100,50,30], contending=[t,f,t](B 弃) → 主池层 [0,1,2] eligible 只 [0,2];边池层 [0];[0] 独拿层
4. 全员同额 → 1 个 pot
5. Distribute 平局余数:pot 100, winners [1,3] → 各 50,无余数
6. Distribute 不平局余数:pot 101, winners [1,3] → 各 50 + 1 给 seat 1(顺位最先)→ [51, 50]
7. Distribute 空赢家通吃:pot 50 eligible [0] winners [] → result[0] += 50
